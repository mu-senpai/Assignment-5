Frequently Asked Questions

What is the Document Object Model (DOM)?

The DOM is a programming interface for web documents. It represents the page structure as a tree of objects, allowing JavaScript to access and manipulate HTML elements dynamically.

How do you select an element from the DOM?

You can select elements using methods like document.getElementById(), document.querySelector(), or document.getElementsByClassName() to target specific elements by their ID, class, or tag.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation allows you to handle events at a parent element level, capturing events from child elements. It improves performance and simplifies code by reducing the need to attach individual listeners to each child.

How do you manipulate an element's attributes and styles using the DOM?

You can use methods like setAttribute() and getAttribute() for attributes. For styles, you can modify the style property, such as element.style.color = "red" to change an element's appearance.
